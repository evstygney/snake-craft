# Snake Craft

Классическая Snake-игра на чистом HTML, CSS и JavaScript.

## Что внутри
- движение по сетке 16x16
- рост змейки после еды
- счёт и лучший результат
- пауза и рестарт
- управление с клавиатуры и экранные кнопки
- базовые тесты для игровой логики

## Запуск локально
Из корня проекта:

```powershell
python -m http.server 4173
```

Затем откройте в браузере:

`http://localhost:4173/index.html`

## Управление
- `Стрелки` или `WASD` - движение
- `Space` - пауза / продолжить
- `R` - начать заново

## Тесты
Если установлен Node.js:

```powershell
node --test .\tests\snake-game.test.mjs
```

## Структура
- `index.html` - страница игры
- `styles.css` - оформление интерфейса
- `src/snake-game.mjs` - чистая игровая логика
- `src/main.js` - рендер и обработка управления
- `tests/snake-game.test.mjs` - тесты логики

## Ручная проверка
- змейка двигается по клеткам без рывков
- нельзя развернуться на 180 градусов в себя
- еда появляется только в свободной клетке
- после еды счёт растёт, змейка увеличивается
- при столкновении со стеной или собой игра заканчивается
- пауза и рестарт работают с клавиатуры и кнопок
